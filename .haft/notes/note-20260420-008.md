# CLI commands task subsumed: skill discovery (prob-20260417-006) + model routing (prob-20260420-003) cover the surface

## Rationale

Vault task 2026-04-17-telegram-cli-commands.md proposed bot-level middleware for /model, /compact, /status, /tasks, /search, /vault-stats. Reviewed against active problem cards: (1) prob-20260417-006 already frames skill discoverability via Telegram slash commands — once solved, /compact, /h-status, /tasks etc. are invocable as skills directly from the BotFather command list with autocomplete. (2) prob-20260420-003 already frames model-tier switching, but as auto-routing (no user command needed) — the /model shortcut is implicit in that solution. (3) No remaining cases require dedicated bot-level command middleware beyond what these two problems address. Decision: close the vault task as subsumed. Do NOT create a new ProblemCard — it would overlap prob-20260417-006 without adding distinct scope.

prob-20260417-006 acceptance criterion: 'User types / in Telegram and sees a browsable list of available skills with one-line descriptions. Tapping one sends the correct invocation to Claude Code.'
