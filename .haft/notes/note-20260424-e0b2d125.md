# Scribe reminder infrastructure: reminder_date detection + fs.watch pickup

## Rationale

Scribe SKILL.md step 2 now detects time references ("tomorrow", "in N days", specific dates) and extracts a reminder_date. Step 6 writes a one-shot scribe_reminder entry to bot-data/schedules.json via bun -e script. Scheduler picks this up in seconds via fs.watch(SCHEDULES_FILE) + reloadOneShots() — no bot restart needed. registeredOneShotIds Set prevents duplicate timer registration on re-reads. Alternative: have Scribe call a bot HTTP endpoint — rejected, no HTTP server in bot, write-to-file is simpler and already supported.

## Affected Files

- `~/.claude/skills/scribe/SKILL.md`
- `src/scheduler.ts`
