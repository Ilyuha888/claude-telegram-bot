# claude-opus-4-7 set as default model via vault project settings

## Rationale

Rather than complex auto-routing (Opus plan / Sonnet execute / Haiku cheap), user chose Opus 4.7 as the single tier for all personal assistant sessions. Implemented by adding "model": "claude-opus-4-7" to /home/assistant/repos/my_obsidian_knowledge_base/.claude/settings.json. This is the right location because CLAUDE_WORKING_DIR points to the vault, so the bot's agent sessions load it via settingSources=["user","local","project"]. Admin PKM sessions (Scribe, Retriever, Curator) opened in the vault also benefit. settings.json (not settings.local.json) so it's committed and applies across machines.
